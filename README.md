# Pizza Sales SQL Analysis Project

## Overview
This project contains SQL queries used to analyze a pizza sales dataset using MySQL.
The analysis focuses on calculating revenue, identifying top-selling pizzas, ranking products, and analyzing category-wise performance.

## Concepts Used
* Joins
* Aggregate Functions
* Window Functions
* Subqueries
* Ranking Functions (RANK, DENSE_RANK)
* Revenue calculations (price × quantity)

## Analysis Performed
* Total revenue calculation
* Top 3 pizzas by revenue
* Revenue by category
* Percentage contribution of each pizza type
* Ranking pizzas within each category
* Running averages using window functions
